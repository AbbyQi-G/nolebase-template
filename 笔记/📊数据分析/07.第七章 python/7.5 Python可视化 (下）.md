- figure概念：整个图表区域称为figure，包含所有可视化元素
- title属性：图表顶部的标题文字，用于说明图表主题
- 坐标轴标签：
    - xlabel：x轴下方的说明文字
    - ylabel：y轴左侧的说明文字
- 图例(legend)：说明图表中不同颜色/样式的数据系列代表什么
- 所属关系：
    - 单坐标系图表中，title/label属于figure或坐标系均可
    - 多坐标系图表中，必须明确属于特定坐标系
